## Overview

**Advent of Code** is an annual coding challenge held in December, offering a series of small programming puzzles. Each puzzle encourages problem-solving, logical thinking, and learning new programming concepts in a fun way.  

- Language used: **Python**  
- Concepts practiced: **loops, conditions, strings, lists, iteration**  
- Focus: building logic, testing programming skills, and having fun
